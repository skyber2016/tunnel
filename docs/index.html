Features

Everything you need, nothing you don't

Built for developers who SSH into jump hosts every day.

⚡

Native AOT — Zero Runtime

Compiled to a native binary with .NET 9 AOT. No runtime installation required. Cold start under 50ms.

🔀

Multi-Port Forwarding

Forward unlimited ports through a single SSH jump host connection simultaneously.

🔄

Background Daemon

Runs as a Systemd user service. Tunnels stay alive even after you close the terminal.

📋

Profile Management

Save multiple SSH tunnel configurations. Switch between dev, staging, and production in seconds.

🔒

Private Key Auth

Connects using your SSH private key. Supports custom key paths per profile.

⬆️

Self-Update

One command to update both CLI and daemon from GitHub Releases. Atomic binary swap with auto-restart.

🖥️

Rich Terminal UI

Powered by Spectre.Console. Beautiful tables, spinners, and progress bars — not just plain text.

🏗️

ARM64 Support

Native binaries for both x86_64 and aarch64. Works on Apple Silicon Macs in Linux VMs too.

🛡️

Systemd Hardening

Service runs with NoNewPrivileges, PrivateTmp, and ProtectSystem for minimal attack surface.

Commands

Full command reference

All 7 commands designed for fast, intuitive use.

tunnel new <name>
Create a new SSH profile interactively
tunnel add <profile> --local N --remote N
Add a port forwarding mapping to a profile
tunnel list
List all saved profiles in a table view
tunnel use <name>
Activate a profile and start the SSH tunnel
tunnel stop
Stop the currently active tunnel
tunnel status
Show live status of each port forwarding connection
tunnel update [--daemon-only]
Self-update to the latest release from GitHub
Uninstall

Remove old versions first

Before installing or reinstalling, you must remove any existing installation. The installer will block with an error if conflicting binaries are found.

⚠️

Conflict check enforced

The unofficial or previous versions of tunnel and tunnel-daemon you may have installed will conflict with a new release. You must uninstall these packages before installing a new version.

Keep profiles
Removes binaries & service. Keeps ~/.tunnel/ profiles.
$ curl -sSL .../uninstall.sh | bash
Full purge
Removes everything including all profile data in ~/.tunnel/.
$ curl -sSL .../uninstall.sh | bash -s -- --purge

What gets removed:

/usr/local/bin/tunnel
CLI binary — always removed
/usr/local/bin/tunnel-daemon
Daemon binary — always removed
~/.config/systemd/user/tunnel.service
Systemd unit file — always removed
~/.tunnel/profiles.json
Profile data — preserved by default, removed only with --purge
